¿Cuál es la diferencia entre una solicitud y una respuesta?

Una solicitud es un problema de enrutamiento de vehículos (VRP) que Fleet Routing puede resolver. Consiste en uno o más vehículos y uno o más envíos para entregar.

Una respuesta es lo que produce la API de Fleet Routing en respuesta a una solicitud para resolver. Las respuestas son visibles en FleetRouting App a través del diagrama de Gantt, en un mapa, o en la vista de Metadatos.
